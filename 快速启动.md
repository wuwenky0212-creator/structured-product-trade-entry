# 快速启动指南

## 🚀 项目已启动！

### 访问地址

在浏览器中打开以下地址：

```
http://localhost:3000
```

### 启动命令

如果需要重新启动服务器，请在 PowerShell 中运行：

```powershell
cmd /c npm run dev
```

或者直接双击运行：`启动项目.bat`

### 服务器状态检查

检查服务器是否在运行：
```powershell
netstat -ano | findstr :3000
```

如果看到 `LISTENING` 状态，说明服务器正在运行。

### 停止服务器

在运行服务器的终端窗口中按 `Ctrl + C`

### 常用命令

```bash
# 启动开发服务器
cmd /c npm run dev

# 构建生产版本
cmd /c npm run build

# 预览生产版本
cmd /c npm run preview

# 代码检查
cmd /c npm run lint
```

### 注意事项

- 如果端口 3000 被占用，Vite 会自动尝试其他端口
- 修改代码后，页面会自动热更新
- 如果遇到问题，请查看终端输出的错误信息
