应用无法访问 - 问题排查步骤
========================================

问题：访问网站显示 "Application failed to respond"

可能的原因：
1. 应用没有正确启动
2. 端口配置不对
3. 启动命令有问题

解决步骤：
========================================

步骤 1：查看部署日志
----------------------------------------
1. 在 Railway 中，点击 "Deployments" 标签
2. 找到最新的部署（应该显示 "ACTIVE"）
3. 点击 "View logs" 或 "Deploy Logs"
4. 查看最后的错误信息

步骤 2：检查启动命令
----------------------------------------
在 Railway Settings > Deploy 部分，确认：
- Start Command: npm start

步骤 3：检查端口配置
----------------------------------------
确保应用使用 Railway 提供的 PORT 环境变量

步骤 4：查看构建日志
----------------------------------------
在 Deployments 中查看 "Build Logs"，确认构建是否成功

常见问题：
========================================
1. 端口不对 - Railway 会自动设置 PORT 环境变量
2. 启动命令错误 - 应该是 npm start
3. 构建失败 - 查看 Build Logs
