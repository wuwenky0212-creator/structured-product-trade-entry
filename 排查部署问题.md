# 部署问题排查指南

## 问题：网站无法正常访问

### 需要检查的地方：

1. **查看部署日志**
   - 在 Railway 中点击 "Deployments" 标签
   - 找到最新的部署，点击 "View logs" 或 "Deploy Logs"
   - 看看最后的错误信息是什么

2. **查看构建日志**
   - 在 "Deployments" 中查看 "Build Logs"
   - 确认构建是否成功
   - 看看有没有错误信息

3. **检查启动命令**
   - 在 Settings > Deploy 中
   - 确认 Start Command 是否正确

4. **检查端口配置**
   - 在 Settings > Networking 中
   - 确认端口是否为 3000

### 可能的问题和解决方案：

#### 问题1：serve 命令找不到
**解决方案**：使用 npx serve

#### 问题2：端口配置错误
**解决方案**：确保使用 Railway 的 PORT 环境变量

#### 问题3：dist 文件夹不存在
**解决方案**：确认构建成功生成了 dist 文件夹
